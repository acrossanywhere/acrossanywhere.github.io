Python的内存模型

a=3 #a初始值为3
b=a #b指向的是a指向的地址，通俗来说，在内存中有一个地址，里面值是3，在操作系统引用的时候，可以用别名a，或者别名b来引用
a=5 #修改a以后b的值不会改变
这里我们用python内置函数 id查看变量的内存地址

结果：
    ('a id ', 38792248, 3)
    ('b id ', 38792248, 3)
    ('a id ', 38792200, 5)
可以看出变量a和变量b是指向同一个地址，别名a指向5，而5又在其他内存区域，所以最后打印a指向的地址发生了变化
